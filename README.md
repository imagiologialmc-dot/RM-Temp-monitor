# Aliva Saúde — Monitor Sala Técnica RM

App PWA para registo de temperatura e humidade da sala técnica de Ressonância Magnética.

## Como publicar no GitHub Pages

1. Cria um repositório público no GitHub chamado `aliva-rm-monitor`
2. Faz upload de todos estes ficheiros para o repositório
3. Vai a Settings > Pages > Source: Deploy from a branch > main > / (root) > Save
4. Em ~2 minutos a app fica em: https://SEU_USUARIO.github.io/aliva-rm-monitor/sala-rm-temperatura-humidade.html

## Instalar no Android
- Abre o link acima no Chrome Android
- Menu (3 pontos) > Instalar app / Adicionar ao ecrã inicial
